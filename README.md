# Slippi Game Retriever
Copies the latest .slp file from the autosave folder to a directory of your choosing.

## Installation
-download SlippiGameRetriever.exe and the three accompanying text files and place them anywhere (so long as they are in the same folder)
-locate folder to which .slp files are automatically saved, copy its full path, open autosaveFolder.txt and paste this path into the file, ensure the file is only one line long and contains no extra characters
-copy the full path of the folder to which you would like selected replays to be sent, open culledFolder.txt and paste this path into the file, ensure the file is only one line long and contains no extra characters
-copy the full path of Slippi (including Dolphin.exe), open SlippiPath.txt and paste this path into the file, ensure the file is only one line long and contains no extra characters
